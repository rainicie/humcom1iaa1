FIRST GRADING

WEB BASICS

As I embarked on my web design journey, the first grading period introduced me to the backbone of every website: HTML. It felt like learning the language of the internet—structuring content, defining elements, and creating the skeletal framework of web pages. The simplicity of HTML was deceptive because, behind it, lay the power to organize information in a way that users could understand and interact with.


2
MIDTERMS

THE ART OF STYLE

After mastering the structure, I moved on to the art of web design: CSS. It was exhilarating to see the dull, lifeless HTML pages evolve into visually appealing, stylish masterpieces. Colors, typography, spacing, and positioning turned a rigid framework into an engaging user experience. At times, the endless tweaking and adjusting felt like solving a puzzle—one line of code could drastically change an entire layout.


3
FINALS

BRINGING PAGES TO LIFE

In the final term, the magic truly began—JavaScript. If HTML was the skeleton and CSS the skin, JavaScript was the heartbeat, breathing life into my websites. Buttons became interactive, elements responded dynamically, and websites evolved into living, functional tools rather than static pages.
